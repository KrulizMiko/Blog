## 🎯 Требование 1: ER-диаграмма БД

**Статус**: ✅ ВЫПОЛНЕНО

### Что создано:
- ✅ Модели БД в SQLAlchemy:
  - `User` - пользователи
  - `Post` - посты
  - `Comment` - комментарии
  - `Category` - категории
  - `Tag` - теги
  - `post_tags` - связь Many-to-Many

### Структура:
```
Users (1) ──┐
            ├─→ Posts (N) ────────┬─→ Categories (N)
            │                     ├─→ Tags (N) [Many-to-Many]
            └─→ Comments (N) ────→ Posts
```
